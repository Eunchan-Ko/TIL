# 백준 11501번 - 주식
## 문제
[클릭하면 문제 페이지로 이동합니다.](https://www.acmicpc.net/problem/11501)

주식을 사고팔아 수익을 내는 것을 목표로 한다. 주식의 일별 가격이 주어질 때, 최대한 많은 수익을 얻기 위해 어떻게 주식을 사고팔아야 할까?
## 내가 푼 아이디어
뒤로 돌면서 최대값을 찾고, 그 최대값보다 작은 값이 나오면 그 차이만큼 수익을 얻는 방식으로 풀이
```python
t = int(input())

for _ in range(t):
    N = int(input())
    stocks = list(map(int, input().split()))
    total_earn = 0
    max_v = stocks[-1]
    for stock in stocks[::-1]:
        if stock > max_v:
            max_v = stock
        total_earn += max_v - stock
    print(total_earn)

```
