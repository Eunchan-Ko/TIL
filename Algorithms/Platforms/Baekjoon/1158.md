# 백준 1158번 - 요세푸스 문제
[문제 링크](https://www.acmicpc.net/problem/1158)
## 문제
요세푸스 문제는 다음과 같다. 1번부터 N번까지 N명의 사람이 원을 이루며 앉아있고, 양의 정수 K가 주어진다. 이제 순서대로 K번째 사람을 제거한다. 이 과정은 N명의 사람이 모두 제거될 때까지 계속된다. 원에서 사람들이 제거되는 순서를 (N, K)-요세푸스 순열이라고 한다. 예를 들어 (7, 3)-요세푸스 순열은 <3, 6, 2, 7, 5, 1, 4>이다.
## 내가 푼 아이디어
1. 리스트를 사용하여 원을 표현한다. 리스트에서 K-1 번째 사람을 제거하고, 그 다음 K-1 번째 사람을 제거하는 과정을 반복한다.
2. 리스트에서 K-1 번째 사람을 제거할 때, 인덱스가 리스트의 길이를 초과하는 경우에는 인덱스를 리스트의 길이로 나눈 나머지로 수정한다.

```python
n, k = map(int, input().split())
lst = list(range(1, n+1))
idx = 0 + k - 1
ans = []
while lst:
    ans.append(lst.pop(idx))
    idx += k - 1
    n -= 1
    if idx >= n and n > 0:
        idx %= n
print('<', end='')
for i in range(len(ans) - 1):
    print(f'{ans[i]},', end=' ')
print(f"{ans[-1]}>")
```
# 추후 개선할 점
1. 이중연결리스트로 구현하여 리스트에서 K-1 번째 사람을 제거할 때 O(1) 시간에 제거할 수 있도록 개선할 수 있다. 현재는 리스트에서 K-1 번째 사람을 제거할 때 O(n) 시간이 걸린다. 이중연결리스트를 사용하면 K-1 번째 사람을 제거할 때 O(1) 시간에 제거할 수 있다.
