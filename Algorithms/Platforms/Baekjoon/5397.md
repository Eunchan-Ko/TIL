# 백준 5397번 - 키로거
[클릭하면 문제 페이지로 이동합니다.](https://www.acmicpc.net/problem/5397)

## 문제
키로거는 키보드의 키가 눌리는 순서를 기록하는 프로그램이다. 키로거는 다음과 같은 키 입력을 기록할 수 있다.
## 내가 푼 아이디어
스택을 이용해 문제를 풀었다. 1406과 같은 방식으로 왼쪽 스택과 오른쪽 스택을 이용해 커서의 위치를 표현했다. 왼쪽 스택에는 커서 왼쪽에 있는 문자들이 들어가고, 오른쪽 스택에는 커서 오른쪽에 있는 문자들이 들어간다. 명령어에 따라 왼쪽 스택과 오른쪽 스택을 조작하여 최종적으로 왼쪽 스택과 오른쪽 스택을 합쳐서 결과를 출력한다.

```python
tes = int(input())
for _ in range(tes):
    password = input()
    l_stk = []
    r_stk = []
    for item in password:
        if item == '<':
            if l_stk:
                r_stk.append(l_stk.pop())
        elif item == '>':
            if r_stk:
                l_stk.append(r_stk.pop())
        elif item == '-':
            if l_stk:
                l_stk.pop()
        else:
            l_stk.append(item)
    l_stk.extend(r_stk[::-1])
    print(''.join(l_stk))
```
