# 백준 2839번 - 설탕 배달
## 문제
최대한 적은 수의 봉지를 만들어야함. 5kg 봉지와 3kg 봉지가 있음. 정확하게 N kg을 만들 수 없다면 -1 출력.
## 내가 푼 아이디어
5kg 봉지를 최대한 많이 사용한 후, 12kg 미만일 때 직접 계산하는 방식을 사용
```python
n = int(input())

bag = 0
while n > 0:
    if n <= 12:
        if n == 3:
            bag += 1
        elif n == 5:
            bag += 1
        elif n==6:
            bag += 2
        elif n==8:
            bag += 2
        elif n==9:
            bag += 3
        elif n==10:
            bag += 2
        elif n==11:
            bag += 3
        elif n==12:
            bag += 4
        else :
            bag = -1
        n = 0
    if n > 5:
        n -= 5
        bag += 1
print(bag)
```
이 방식은 5kg 봉지를 최대한 많이 사용하기 때문에, 최소 봉지 수를 보장함.
하지만 12kg 미만일 때의 경우의 수를 일일이 다 계산해줘야 하는 단점이 있음.
## 더 나은 아이디어 정리
1. 5kg 봉지를 최대한 많이 사용 = n // 5
2. 남은 무게를 3kg 봉지로 채울 수 있는지 확인 = (n % 5) % 3 == 0
   
컴퓨터는 나눗셈을 잘한다.
