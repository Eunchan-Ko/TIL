# 백준 1182번 - 부분수열 개수
## 문제
N개의 정수로 이루어진 수열이 있을 때, 크기가 양수인 부분수열 중에서 그 수열의 원소를 다 더한 값이 S가 되는 경우의 수를 구하는 프로그램을 작성하시오.
## 내가 푼 아이디어
bit를 인덱스처럼 써서 모든 수열을 다 검사해봤다.

```python
N, S = map(int, input().split())
lst = list(map(int, input().split()))
cnt = 0
for i in range(1<<N):
    idx_on = list(map(int, f"{i:0{N}b}"))
    sum_all = 0
    f = False
    for j in range(len(idx_on)):
        if idx_on[j]:
            f = True
            sum_all += lst[j]
    if f and sum_all == S:
        cnt += 1
print(cnt)
```
