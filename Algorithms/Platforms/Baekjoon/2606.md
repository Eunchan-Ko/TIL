# 백준 2606번 - 바이러스
## 문제
[2606번 - 바이러스](https://www.acmicpc.net/problem/2606)
신종 바이러스인 웜 바이러스는 네트워크를 통해 전파된다. 한 컴퓨터가 웜 바이러스에 걸리면 그 컴퓨터와 네트워크 상에서 연결되어 있는 모든 컴퓨터는 웜 바이러스에 걸리게 된다.

예를 들어 7대의 컴퓨터가 <그림 1>과 같이 네트워크 상에서 연결되어 있다고 하자. 1번 컴퓨터가 웜 바이러스에 걸리면 웜 바이러스는 2번과 5번 컴퓨터를 거쳐 3번과 6번 컴퓨터까지 전파되어 2, 3, 5, 6 네 대의 컴퓨터는 웜 바이러스에 걸리게 된다. 하지만 4번과 7번 컴퓨터는 1번 컴퓨터와 네트워크상에서 연결되어 있지 않기 때문에 영향을 받지 않는다.

어느 날 1번 컴퓨터가 웜 바이러스에 걸렸다. 컴퓨터의 수와 네트워크 상에서 서로 연결되어 있는 정보가 주어질 때, 1번 컴퓨터를 통해 웜 바이러스에 걸리게 되는 컴퓨터의 수를 출력하는 프로그램을 작성하시오.
## 내가 푼 아이디어
각 노드의 연결 정보를 딕셔너리를 이용해 그래프로 표현함.
그리고 이걸 반복문을 돌면서 바이러스 감염 여부를 체크하는 방식으로 풀이함.
```python
n = int(input())
line = int(input())

nodes = {}
virus = [0] * (n + 1)
virus[1] = 1

for _ in range(line):
    node, next = map(int, input().split())
    if nodes.setdefault(node, [next]) != [next]:
        nodes[node].append(next)
for _ in range(n):        
  for key, value in nodes.items():
      is_virus = False
      for i in value:
        if virus[i] == 1:
            is_virus = True
        if virus[key] == 1 or is_virus:
          virus[key] = 1
          for i in value:
              virus[i] = 1
print(sum(virus) - 1)
```
이 방식은 바이러스에 감염된 컴퓨터가 있는지 체크하기 위해 반복문을 여러 번 돌면서 체크하는 방식이기 때문에, 시간 복잡도가 O(n^2)으로 비효율적임.
## 더 나은 아이디어 정리
1. 그래프 탐색 알고리즘인 DFS나 BFS를 이용해서 바이러스에 감염된 컴퓨터를 체크하는 방식으로 풀이할 수 있음.
2. DFS나 BFS는 그래프의 모든 노드를 한 번씩 방문하기 때문에, 시간 복잡도가 O(n)으로 효율적임.
