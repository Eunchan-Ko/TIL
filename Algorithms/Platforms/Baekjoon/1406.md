# 백준 1406번 - 에디터
[클릭하면 문제 페이지로 이동합니다.](https://www.acmicpc.net/problem/1406)

## 문제
한 줄로 된 간단한 에디터를 구현하려고 한다. 이 에디터에는 커서가 있는데, 커서는 문장의 맨 앞(첫 번째 문자 앞)부터 시작한다. 이 에디터에는 다음과 같은 명령어들이 있다.

l_stk = word
r_stk = []
for _ in range(order_num):
    order = list(input().strip().split())
    if order[0] == 'L':
        if l_stk:
            r_stk.append(l_stk.pop())
    elif order[0] == 'D':
        if r_stk:
            l_stk.append(r_stk.pop())
    elif order[0] == 'B':
        if l_stk:
            l_stk.pop()
    else:
        l_stk.append(order[1])
print(''.join(l_stk), end='')
print(''.join(r_stk[::-1]))
