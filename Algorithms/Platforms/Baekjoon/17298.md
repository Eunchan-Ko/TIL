# 백준 17298번 - 오큰수
## 문제
[6198번 - 옥상정원](https://www.acmicpc.net/problem/17298)
크기가 N인 수열 A = A1, A2, ..., AN이 있다. 수열의 각 원소 Ai에 대해서 오큰수 NGE(i)를 구하려고 한다. Ai의 오큰수는 오른쪽에 있으면서 Ai보다 큰 수 중에서 가장 왼쪽에 있는 수를 의미한다. 그러한 수가 없는 경우에 오큰수는 -1이다.

예를 들어, A = [3, 5, 2, 7]인 경우 NGE(1) = 5, NGE(2) = 7, NGE(3) = 7, NGE(4) = -1이다. A = [9, 5, 4, 8]인 경우에는 NGE(1) = -1, NGE(2) = 8, NGE(3) = 8, NGE(4) = -1이다.

## 내가 푼 아이디어
1. 거꾸로 순회하면서 스택의 가장 최근 값을 현재 값 기준 오른쪽에서 큰 수중 제일 먼저인 수로 설정함.

```python
N = int(input())
A = list(map(int, input().split()))
NGE = list([-1] * N)

resent_set = N-1
stack = []
for i in range(N-1, -1, -1):
    if not stack:
        stack.append(A[i])
    if A[i] < stack[-1]:
        NGE[i] = stack[-1]
        stack.append(A[i])
    if A[i] >= stack[-1]:
        while stack and stack[-1] <= A[i]:
            stack.pop()
        if stack:
            NGE[i] = stack[-1]
        stack.append(A[i])
print(*NGE)
```
