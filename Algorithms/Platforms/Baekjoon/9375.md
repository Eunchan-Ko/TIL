# 백준 9375번 - 패션왕 신해빈
## 문제
[클릭하면 문제 페이지로 이동합니다.](https://www.acmicpc.net/problem/9375)

해빈이는 패션에 매우 민감해서 한번 입었던 옷들의 조합을 절대 다시 입지 않는다. 예를 들어 오늘 해빈이가 안경, 코트, 상의, 신발을 입었다면, 다음날은 바지를 추가로 입거나 안경대신 렌즈를 착용하거나 해야한다. 해빈이가 가진 의상들이 주어졌을때 과연 해빈이는 알몸이 아닌 상태로 며칠동안 밖에 돌아다닐 수 있을까?
## 내가 푼 아이디어
item 종류별 개수를 list로 저장한 후 각각의 item 종류별로 (개수 + 1)씩 곱한 후, 마지막에 1을 빼주는 방식으로 풀이
```python
from collections import defaultdict
# test case 수 입력
t = int(input())

for _ in range(t):
    # 의상 수 입력
    n = int(input())
    wear = defaultdict(int)
    for _ in range(n):
        item, key = input().split()
        wear[key] += 1
    # 만들어진 dictionary에서 value값을 list로 만들어서 각각의 item 종류별로 (개수 + 1)씩 곱한 후, 마지막에 1을 빼주는 방식으로 풀이
    sort_num = 0
    for _ in wear.keys():
        sort_num += 1
    sort_lst = [0] * sort_num
    i = 0
    for v in wear.values():
        sort_lst[i] = v+1
        i += 1
    
    answer = 1
    for i in sort_lst:
        answer *= i
    # 1은 아무것도 입지 않은 경우[모든 item을 선택하지 않은 경우]이므로, 1을 빼준다.
    answer -= 1
    print(answer)
```
## 더 나은 아이이어
굳이 list로 만들 필요 없이 바로 dictionary의 value값을 이용해서 각각의 item 종류별로 (개수 + 1)씩 곱한 후, 마지막에 1을 빼주는 방식으로 풀이하면 메모리를 조금 더 절약 할 수 있지 않을까..
