# 백준 1931 - 회의실 배정
## 문제
[클릭하면 문제 페이지로 이동합니다.](https://www.acmicpc.net/problem/1931)

## 내가 푼 아이디어
종료 시간을 기준으로 정렬한 후, 이전 회의가 끝나는 시간을 기준으로 정렬하여 회의실 배정 문제를 풀이하였다.
## 코드
```python
N = int(input())
meetings = []

for _ in range(N):
    s, e = map(int, input().split())
    meetings.append((s, e))

meetings.sort(key=lambda x: (x[1], x[0]))

last_end_time = -1
count = 0

for start, end in meetings:
    if start >= last_end_time:
        count += 1
        last_end_time = end
print(count)
```

# 정리
1. 회의실 배정 문제는 종료 시간을 기준으로 정렬한 후, 이전 회의가 끝나는 시간을 기준으로 정렬하여 풀이할 수 있다.
2. 회의실 배정 문제는 그리디 알고리즘을 사용하여 풀이할 수 있다.
