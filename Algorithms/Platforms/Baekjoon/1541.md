# 백준 1541번 - 잃어버린 괄호
[클릭하면 문제 페이지로 이동합니다.](https://www.acmicpc.net/problem/1541)

## 문제
세준이는 양수와 +, -, 그리고 괄호를 가지고 식을 만들었다. 그리고 나서 세준이는 괄호를 모두 지웠다.

그리고 나서 세준이는 괄호를 적절히 쳐서 이 식의 값을 최소로 만들려고 한다.

괄호를 적절히 쳐서 이 식의 값을 최소로 만드는 프로그램을 작성하시오.
## 내가 푼 아이디어
식을 - 기준으로 쪼개고, 각각의 식을 + 기준으로 쪼개서 더한 후, - 기준으로 쪼갠 식들의 합을 빼주는 방식으로 풀이하였다.

```python
sik = input().split('-')
minus = []
for i in sik:
    sum = 0
    for j in i.split('+'):
        sum += int(j)
    minus.append(sum)

result = minus[0]
for i in range(1, len(minus)):
    result -= minus[i]

print(result)
```
