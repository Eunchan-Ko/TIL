# 백준 2178번 - 미로 탐색
## 문제
[2178](https://www.acmicpc.net/problem/2178)
N×M크기의 배열로 표현되는 미로가 있다. 1은 이동할 수 있는 칸을 나타내고, 0은 이동할 수 없는 칸을 나타낸다. 이러한 미로가 주어졌을 때, (1, 1)에서 출발하여 (N, M)의 위치로 이동하려고 한다. 이때 한 번에 한 칸씩 이동할 수 있으며, 이동할 수 있는 칸으로만 이동해야 한다. 또한, 미로는 반드시 탈출할 수 있는 형태로 제시된다. 탈출하기 위해 움직여야 하는 최소 칸의 개수를 구하는 프로그램을 작성하시오.

## 내가 푼 아이디어
미로를 2차원 리스트로 표현하고, (1, 1)에서 출발하여 (N, M)의 위치로 이동하기 위해 bfs를 사용하여 탐색한다. 

bfs를 사용하여 탐색하면서 이동할 수 있는 칸으로만 이동하도록 한다. 그리고 이동할 때마다 이동한 칸의 개수를 카운트하여 최소 칸의 개수를 구한다.

```python
from collections import deque
n, m = map(int, input().split())
arr = [list(map(int, input())) for _ in range(n)]

delta = [(1, 0), (0, 1), (-1, 0), (0, -1)]
que = deque()
que.append((0, 0))
now = que.popleft()
visited = [[0] * m for _ in range(n)]
visited[0][0] = 1
while now != (m-1, n-1):
    for d in delta:
        dx, dy = now[0] + d[0], now[1] + d[1]
        if 0 <= dx < m and 0 <= dy < n:
            if arr[dy][dx] and visited[dy][dx] == 0:
                que.append((dx, dy))
                visited[dy][dx] = visited[now[1]][now[0]] + 1
    if que:
        now = que.popleft()
print(visited[n-1][m-1])
```
# 정리
1. 미로를 2차원 리스트로 표현하고, (1, 1)에서 출발하여 (N, M)의 위치로 이동하기 위해 bfs를 사용하여 탐색하였다.
2. bfs를 사용하여 탐색하면서 이동할 수 있는 칸으로만 이동하도록 하였다. 그리고 이동할 때마다 이동한 칸의 개수를 카운트하여 최소 칸의 개수를 구하였다.
