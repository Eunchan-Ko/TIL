# 백준 2493 - 탑
## 문제
[2493](https://www.acmicpc.net/problem/2493)
N개의 탑이 수평선상에 일렬로 서 있다. 각 탑은 레이저 송신기와 수신기를 가지고 있다. 모든 탑의 높이는 서로 다르다. 레이저 송신기는 탑의 꼭대기에 설치되어 있고, 레이저 수신기는 탑의 바닥에 설치되어 있다. 레이저 송신기는 왼쪽에서 오른쪽으로 레이저를 발사한다. 레이저는 자신보다 높은 탑에 부딪히면 그 탑에서 반사되어 다시 왼쪽으로 이동한다. 레이저가 다른 탑에 부딪히지 않고 왼쪽으로 나가버리는 경우도 있다. 각 탑에서 발사된 레이저가 처음으로 부딪히는 탑의 번호를 구하는 프로그램을 작성하시오.
## 내가 푼 아이디어
1. 전체탐색
탑의 개수 N과 각 탑의 높이를 입력받는다. 그리고 각 탑의 번호와 높이를 리스트로 저장한다. 그리고 전체탐색을 하면서 거꾸로 가면서 만약에 i - 1의 사이즈가 더 작으면 더 큰걸 만날 때까지 감.
2. stack 사용
    1. 탑의 개수 N과 각 탑의 높이를 입력받는다. 그리고 각 탑의 번호와 높이를 리스트로 저장한다. 
    
    2. 
        1. 그리고 stack을 사용하여 거꾸로 가면서 만약에 i - 1의 사이즈가 더 작으면 더 큰걸 만날 때까지 감.
        2. 앞 건물이 더 높을때, 가장 최근 값을 POP 해서 resent로 저장한 뒤 더 높은 건물을 만날때까지 get_signal에 정보를 저장한다. 
    
    3. while 문이 끝났을 때 가장 최근에 pop 한 정보를 다시 append해서 stack이 비어있지 않도록 한다.

## 코드
```python
N = int(input())
lst = list(map(int, input().split()))
for i in range(1, N+1):
    lst[i-1] = [i, lst[i-1]]

get_signal = [0] * N
# 전체탐색
# for i in range(N-1, 0, -1):
#     # 거꾸로 가면서
#     # 만약에 i - 1의 사이즈가 더 작으면 더 큰걸 만날 때까지 감.
#     for j in range(i-1, -1, -1):
#         if lst[j] >= lst[i]:
#             get_signal[i] = j+1
#             break
# 전체탐색 끝
# stack 사용
stack = [lst[N-1]]
for i in range(N-2, -1, -1):
    # print(stack)
    # 스택에 값이 존재하고, 앞 건물이 더 낮으면 stack에 건물을 append
    if stack and lst[i][1] < stack[-1][1]:
        stack.append(lst[i])
    # 앞 건물이 더 높을때,
    else:
        # 가장 최근 값을 POP 해서 resent로 저장한 뒤
        # 더 높은 건물을 만날때까지 get_signal에 정보를 저장한다.
        resent = stack.pop()
        while lst[i][1] >= resent[1]:
            get_signal[resent[0]-1] = i + 1
            if stack:
                resent = stack.pop()
            else:
                # 최대 값보다 크게 만들어서 오류가 나지 않도록하고, while 문이 끝날 수 있도록 함.
                resent = [500001, 1000000001]
        else:
            if resent[1] != 1000000001:
                # while 문이 끝났을 때 가장 최근에 pop 한 정보를 다시 append.
                stack.append(resent)
        stack.append(lst[i])
# stack 끝
print(*get_signal)
```

# 정리
1. 전체탐색과 stack을 사용하여 풀이하였다.
2. 전체탐색은 O(N^2)으로 시간초과가 발생하였고, stack을 사용하여 O(N)으로 풀이하였다.
