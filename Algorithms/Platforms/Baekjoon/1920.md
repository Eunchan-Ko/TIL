# 백준 1920번 - 수 찾기
## 문제
N개의 정수 A[1], A[2], ..., A[N]이 주어져 있을 때, 이 안에 X라는 정수가 존재하는지 알아내는 프로그램을 작성하시오.
## 내가 푼 아이디어
`.count()` 메서드를 사용하여 리스트에 해당 숫자가 있는지 확인
```pythonn = int(input())
a = list(map(int, input().split()))
m = int(input())
b = list(map(int, input().split()))
for i in b:
    if a.count(i) > 0:
        print(1)
    else:
        print(0)
```
`.count()` 메서드는 리스트에서 특정 값의 개수를 세어주는 메서드이다.

그런데 시간초과가 났음. 이 문제를 어떻게 해결할까 고민하다가 `set()` 자료형을 사용하면 시간복잡도를 줄일 수 있다는 것을 알게 되었음.
```python
n = int(input())
a = set(map(int, input().split()))
m = int(input())
b = list(map(int, input().split()))
for i in b:
    if i in a:
        print(1)
    else:
        print(0)
```
`set()` 자료형은 중복된 값을 허용하지 않고, 해시 테이블을 사용하여 데이터를 저장하기 때문에 평균적으로 O(1)의 시간복잡도로 탐색이 가능하다. 따라서 리스트에서 `.count()` 메서드를 사용하는 것보다 훨씬 효율적이다.

## 정리
1. 리스트에서 특정 값의 개수를 세는 `.count()` 메서드는 시간복잡도가 O(n)이다.
2. `set()` 자료형을 사용하면 평균적으로 O(1)의 시간복잡도로 탐색이 가능하다.
3. 따라서 많은 데이터를 다룰 때는 `set()` 자료형을 사용하는 것이 효율적이다.
